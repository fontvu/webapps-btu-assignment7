A Minimal effort Web App built with Plain JS and Firebase

This app, which is accompanied by a tutorial, is the 1st example app of our incremental sequence of six examples of Plain JavaScript and Firebase Web Apps. This app supports the following operations:

  • Update a customer record
  • Update a product record
  • Update a product catalog record
  • Update a event record
