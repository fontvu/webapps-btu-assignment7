Minimal App — Public Library

A Minimal effort Web App built with Plain JS and Firebase

This app, which is accompanied by a tutorial, is the 1st example app of our incremental sequence of six examples of Plain JavaScript and Firebase Web Apps. This app supports the following operations:

  • Create a new customer record
  • Retrieve and list all customer records
  • Update a customer record
  • Delete a customer record
    •

    For playing with the app, you may first want to and then view it with . For observing the internal status and error messages, turn on the JavaScript console.

    This minimal version of our JavaScript and Firebase data management application only includes a minimum of the overall functionality required for a complete app. It takes care of only one object type ("customers") and supports the four standard data management operations (Create / Retrieve / Update /Delete), but it needs to be enhanced by styling the user interface with CSS rules and by adding further important parts of the app's overall functionality, as explained in our other Plain JS Front-End Web Apps with JavaScript tutorials.
